## Features

- 借鉴React-SPA作者的开发方式
- 采用ES6语法，组件化、模块化代码功能
- 前端控制路由

```bash
$ git clone git@github.com:ifeng2016/Agency-react.git <yourAppName>
$ cd <yourAppName>
$ npm install
```

Then, launch the project app.
```bash
$ cd app
$ webpack-dev-server
```
Visit http://127.0.0.1:8080 with your browser.
